Absolute Difference (Abs Diff):
Formula: |Actual − Predicted|
This measures the raw error between the actual and predicted values. Smaller differences indicate higher accuracy.

Percentage Difference (Pct Diff):
Formula: (|Actual − Predicted| / Actual) × 100%
Provides a relative error based on the actual value, which is useful when values vary greatly.

Mean Squared Error (MSE):
Formula: (1/n) Σ (Actual − Predicted)²
Penalizes larger errors more strongly and gives an average error measure.

Root Mean Squared Error (RMSE):
Formula: √MSE
On the same scale as the target variable; lower RMSE indicates better predictions.

R-Squared (R²):
Formula: 1 - [Σ (Actual − Predicted)² / Σ (Actual − Actual_mean)²]
Indicates the proportion of variance explained by the model; values closer to 1 are better.
